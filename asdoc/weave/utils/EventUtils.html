doubleBind()method 
public static function doubleBind(primaryHost:IEventDispatcher, primaryProperty:String, secondaryHost:IEventDispatcher, secondaryProperty:String):void

This function sets up a two-way binding. Upon calling this function, the value from the primary host's property will be copied to the secondary host's property.

Parameters
primaryHost:IEventDispatcher — The first host.
 
primaryProperty:String — The name of the property on the first host.
 
secondaryHost:IEventDispatcher — The second host.
 
secondaryProperty:String — The name of a property on the second host.
generateDelayedCallback()method 
public static function generateDelayedCallback(relevantContext:Object, callback:Function, delay:int = 500, passDelayedParameters:Boolean = false):Function

This function generates a delayed version of a callback.

Parameters
relevantContext:Object — If this is not null, then the callback will be removed when the relevantContext object is disposed via SessionManager.dispose(). This parameter is typically a 'this' pointer.
 
callback:Function — The callback function.
 
delay:int (default = 500) — The number of milliseconds to delay before running the callback.
 
passDelayedParameters:Boolean (default = false) — If this is set to true, the most recent parameters passed to the delayed callback will be passed to the original callback when it is called. If this is set to false, no parameters will be passed to the original callback.

Returns
Function — A wrapper around the callback that remembers the parameters and delays calling the original callback.
