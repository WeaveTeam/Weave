Classpublic class AbstractSimplifiedPlotter
InheritanceAbstractSimplifiedPlotter AbstractPlotter
SubclassesBarChartPlotter, PieChartPlotter_old, ScatterPlotPlotter, SimpleTextGlyphPlotter

The purpose of this class is to simplify the session state of an IPlotter object. For example, a Histogram or a Bar Chart can be created by simplifying a RectanglePlotter, hiding the hard-coded part of the session state and making only a few sessioned properties public.
